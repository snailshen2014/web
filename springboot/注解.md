### @Value VS @ConfigurationProperties



|                                                 | @ConfigurationProperties     | @Value                 |
| ----------------------------------------------- | ---------------------------- | ---------------------- |
| 功能                                            | 批量注入配置文件中的属性到类 | 单个注入属性到类的属性 |
| 松散语法(cluster-name,clusterName,cluster_name) | 支持                         | 不支持                 |
| SpEL(#{1*2})                                    | 不支持                       | 支持                   |
| JSR303数据校验(@Validated)                      | 支持                         | 不支持                 |
| 复杂数据类型封装(Map,List)                      | 支持                         | 不支持                 |
